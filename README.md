Repository for tahatsahin.com

This repo has 2 different services. ExpressJs Frontend Service and Golang Backend.

tahatsahin.com is my personal website, hosted on AWS.

Used technologies:

Frontend: ExpressJs, AWS CloudFront, AWS S3 Bucket
Backend: Golang, AWS API Gateway, AWS Lambda, AWS Simple Email Service (SES)
Other: 
  - AWS Route53 for DNS
  - AWS Certificate Manager for SSL
  - Github Actions for CI/CD, both backend and frontend
